# Folder na obrazy

Tutaj wgraj:
- Logo firmy (np. `logo.png`)
- Zdjęcia do galerii
- Zdjęcia tła (jeśli będą)

## Rekomendowane rozmiary:

- **Logo:** 200-400px szerokości, format PNG z przezroczystym tłem
- **Hero background:** 1920x1080px, format JPG
- **Zdjęcia w sekcjach:** 800-1200px szerokości, format JPG

## Optymalizacja obrazów:

Przed wgraniem zoptymalizuj obrazy używając:
- [TinyPNG](https://tinypng.com/) - kompresja PNG/JPG
- [Squoosh](https://squoosh.app/) - zaawansowana kompresja
